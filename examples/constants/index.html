Swift by example constants
Declare constants with `let`.
A constant can appear anywhere a `var` statement can. 
let s = "string"
println(s)            // string

let n = 500000000
println(n)            // 500000000

let r = 3 ^ 20 / n
println(r)            // 3
'let' declarations require an initializer expression
